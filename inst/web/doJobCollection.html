Execute Jobs of a JobCollection

Usage

doJobCollection(jc, con = stdout())

Arguments

jc
[JobCollection] Either an object of class “JobCollection” as returned by makeJobCollection or a string point to file containing a “JobCollection” (saved with saveRDS).
con
[connection | character(1)] A connection for the output. Defaults to stdout. Alternatively the name of a file to write to.

Value

[data.table]. Data table with updates on the computational status, i.e. a table with the columns “job.id”, “started” (unix time stamp of job start), “done” (unix time stamp of job termination), “error” (error message as string) and memory (memory usage as double).

Description

Executes every job in a JobCollection. This function is intended to be called on the slave.
