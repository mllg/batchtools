Map Operation for Batch Systems

Usage

batchMap(fun, ..., more.args = list(), reg = getDefaultRegistry())

Arguments

fun
[function] Function to map over ....
...
[any] Arguments to vectorize over (list or vector).
more.args
[list] A list of other arguments passed to fun. Default is an empty list.
reg
[Registry] Registry. If not explicitly passed, uses the last created registry.

Value

[data.table]. Generated job ids are stored in the column “job.id”. See JoinTables for examples on working with job tables.

Description

A parallel Map for batch systems. Note that this function only defines the jobs. The actual computation is started with submitJobs and results can be collected with reduceResultsList, reduceResults or loadResult. For a synchronous Map-like execution see btmapply.

Examples

reg = makeRegistry(file.dir = NA, make.default = FALSE)


Sourcing configuration file '~/.batchtools.conf.R' ...

f = function(x, y) x^2 + y
ids = batchMap(f, x = 1:10, more.args = list(y = 100), reg = reg)


Adding 10 jobs ...

print(ids)
