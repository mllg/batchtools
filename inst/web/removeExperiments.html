Remove Experiments

Usage

removeExperiments(ids = integer(0L), reg = getDefaultRegistry())

Arguments

ids
[data.frame or integer] A data.frame (or data.table) with a column named “job.id”. Alternatively, you may also pass a vector of integerish job ids. If not set, defaults to no job.
reg
[ExperimentRegistry] Registry. If not explicitly passed, uses the last created registry.

Value

[data.table] of removed job ids.

Description

Remove Experiments from an ExperimentRegistry.
