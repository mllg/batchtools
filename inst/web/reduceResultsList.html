Apply Functions on Results

Usage

reduceResultsList(ids = NULL, fun = NULL, ..., reg = getDefaultRegistry())


reduceResultsDataTable(ids = NULL, fun = NULL, ..., fill = FALSE, reg = getDefaultRegistry())

Arguments

ids
[data.frame or integer] A data.frame (or data.table) with a column named “job.id”. Alternatively, you may also pass a vector of integerish job ids. If not set, defaults to the return value of findDone.
fun
[function] Function to apply to each result. The result is passed unnamed as first argument. If NULL, the identity is used. If the function has the formal argument “job”, the Job/Experiment is passed to the function.
...
[ANY] Additional arguments passed to to function fun.
reg
[Registry] Registry. If not explicitly passed, uses the last created registry.
fill
[logical(1)] In reduceResultsDataTable: This flag is passed down to rbindlist which is used to convert the results to a data.table.

Value

reduceResultsList returns a list, reduceResultsDataTable returns a data.table with columns “job.id” and additional result columns as returned by (see rbindlist).

Description

Applies a function on the results of your finished jobs and thereby collects them in a list or data.table.
