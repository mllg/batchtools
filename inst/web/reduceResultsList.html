Aggregate Results

Usage

reduceResultsList(ids = NULL, fun = NULL, ..., reg = getDefaultRegistry())


reduceResultsDataTable(ids = NULL, fun = NULL, ..., fill = FALSE, reg = getDefaultRegistry())

Arguments

ids
[data.frame or integer] A data.frame (or data.table) with a column named “job.id”. This is usually the result of a findJobs operation, but you may also pass any other table here, e.g. the return of getJobInfo. Alternatively, you may also pass a vector of integerish job ids. If not set, defaults to result of findDone.
fun
[function] Function to apply to each result. The results are passed unnamed as first argument.
...
[ANY] Additional arguments passed to to function fun.
reg
[Registry] Registry. If not explictitly passed, uses the last created registry.
fill
[logical(1)] In reduceResultsDataTable: This flag is passed down to rbindlist which is used to convert the results to a data.table.

Value

reduceResultsList returns a list, reduceResultsDataTable returns a data.table with columns “job.id” and “result”.

Description

Functions to conveniently fetch the results from a Registry.
