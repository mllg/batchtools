Grep Log Files for a Pattern

Usage

grepLogs(ids = NULL, pattern, reg = getDefaultRegistry())

Arguments

ids
[data.frame or integer] A data.frame (or data.table) with a column named “job.id”. Alternatively, you may also pass a vector of integerish job ids. If not set, defaults to the return value of findSubmitted.
pattern
[character(1L)] Regular expression.
reg
[Registry] Registry. If not explicitly passed, uses the last created registry.

Value

[data.table]. Matching job ids are stored in the column “job.id”.

Description

Greps through log files and reports jobs with matches.
