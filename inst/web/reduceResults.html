Reduce Results

Usage

reduceResults(fun, ids = NULL, init, ..., reg = getDefaultRegistry())

Arguments

fun
[function] A function to reduce the results. The result of previous iterations (or the init) will be passed as first argument, the result of of the i-th iteration as second. See Reduce for some examples. If the function has the formal argument “job”, the job description is passed.
ids
[data.frame or integer] A data.frame (or data.table) with a column named “job.id”. This is usually the result of a findJobs operation, but you may also pass any other table here, e.g. the return of getJobInfo. Alternatively, you may also pass a vector of integerish job ids. If not set, defaults to result of findDone.
init
[ANY] Initial element, as used in Reduce. Default is the first result.
...
[ANY] Additional arguments passed to to function fun.
reg
[Registry] Registry. If not explicitly passed, uses the last created registry.

Value

Aggregated results, return type depends on function. If ids is empty, reduceResults returns init (if available) or NULL otherwise.

Description

A version of Reduce for Registry objects.

Examples

reg = makeTempRegistry()
batchMap(function(x) x^2, x = 1:10, reg = reg)
