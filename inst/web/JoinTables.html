Inner, Left, Right, Outer and Anti Join for Job Tables

Usage

ijoin(x, y)


ljoin(x, y)


rjoin(x, y)


ojoin(x, y)


ajoin(x, y)

Arguments

x
[data.table] Data table with key “job.id” as returned by most functions in batchtools.
y
[data.table] Data table with key “job.id” as returned by most functions in batchtools.

Value

[data.table] with key “job.id”. See JoinTables for examples on working with job tables.

Description

These helper functions perform join operations on job tables. They are basically one-liners with additional argument checks for sanity. See http://rpubs.com/ronasta/join_data_tables for a overview of join operations in data table or alternatively dplyr's vignette on two table verbs.

# create two tables for demonstration
reg = makeRegistry(file.dir = NA, make.default = FALSE)

x = getJobPars(reg = reg)
y = findJobs(x >= 2 & x <= 5, reg = reg)
y$extra.col = head(letters, nrow(y))

# inner join: similar to intersect() on ids, keep all columns of x and y
ijoin(x, y)

# left join: use all ids from x, keep all columns of x and y
