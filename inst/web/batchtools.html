The batchtools Package

Description

For bug reports and feature requests please use the tracker: https://github.com/mllg/batchtools.

Package options

batchtools.compress
Compression to use in saveRDS which is used to write the Registry, JobCollection jobs and result files. For very slow file systems, set to “xz” to achieve less IO at the expense of more CPU cycles. Defaults to TRUE (“gzip” compression).

batchtools.conf.file
Default path to file which will be sourced while creating a Registry. Can be used to set system-wide ClusterFunctions or default resources.

batchtools.temp.dir
Temporary directory to use in makeTempRegistry. On some cluster systems you might want to set this to a directory which is shared between nodes. Default is the system temp directory as returned by tempdir.

batchtools.verbose
Verbosity. Set to FALSE to suppress info messages and progress bars.
