Abstract base class constructor for general workers

Usage

makeWorker(nodename, ncpus = 0L, max.jobs = NULL, max.load = NULL, debug = FALSE)

Arguments

nodename
[character(1)] Host name of node.
ncpus
[integers(1)] Number of VPUs of worker. Default (0) means to query the worker.
max.jobs
[integer(1)] Maximal number of jobs that can run concurrently for the current registry. Default is ncpus.
max.load
[numeric(1)] Load average (of the last 5 min) at which the worker is considered occupied, so that no job can be submitted. Default is ncpus.
debug
[logical(1)] Print some verbose info messages. Default is FALSE.

Value

[Worker].

Description

Functions to construct local and remote workers.

Examples

## Not run: 
# # create a worker for the local machine and use 4 CPUs.
# makeWorker("localhost", ncpus = 4, max.jobs = 4)
# ## End(Not run)
