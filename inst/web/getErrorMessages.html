Retrieve error messages

Usage

getErrorMessages(ids = NULL, missing.as.error = FALSE, reg = getDefaultRegistry())

Arguments

ids
[data.frame or integer] A data.frame (or data.table) with a column named “job.id”. Alternatively, you may also pass a vector of integerish job ids. If not set, defaults to the return value of findError.
missing.as.error
[logical(1)] Treat missing results as errors? If TRUE, an error message is imputed for jobs which are not terminated. Default is FALSE
reg
[Registry] Registry. If not explicitly passed, uses the last created registry.

Value

[data.table]. Table with columns “job.id”, “terminated” (logical), “error” (logical) and “message” (string).

Description

Extracts error messages from the internal data base and returns them in a compact table. For further investigation, see showLog.
