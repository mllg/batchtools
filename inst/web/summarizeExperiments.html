Quick Summary over Experiments

Usage

summarizeExperiments(ids = NULL, by = c("problem", "algorithm"), reg = getDefaultRegistry())

Arguments

ids
[data.frame or integer] A data.frame (or data.table) with a column named “job.id”. Alternatively, you may also pass a vector of integerish job ids. If not set, defaults to all jobs.
by
[character] Split the resulting table by columns of getJobPars.
reg
[ExperimentRegistry] Registry. If not explicitly passed, uses the last created registry.

Value

[named list] with elements “table” (data.table] of frequencies), “problems” (character of problem names) and “algorithms” (character of algorithm names).

Description

Returns a frequency table of defined experiments.
